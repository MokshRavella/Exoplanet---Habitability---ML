# Exoplanet Habitability Analysis using Machine Learning

This project analyzes exoplanet data to predict whether a planet could be habitable.

## Technologies Used
- Python
- Colab Notebook
- Machine Learning
- Pandas
- Scikit-learn

## Project Overview
The model analyzes planetary parameters such as:
- Mass
- Radius
- Temperature
- Orbital characteristics

Using ML algorithms to classify potentially habitable planets.

## How to Run
1. Download the notebook
2. Open in Colab Notebook
3. Run all cells
